# FlatOut: Ultimate Carnage Most Wanted Physics

Experiment for FlatOut: Ultimate Carnage that converts the physics and handling model from NFS Most Wanted and Carbon

Massive thanks to everyone involved with [dbalatoni13's MW reverse engineering project](https://github.com/dbalatoni13/nfsmw), this mod was heavily based off of it.

## Installation

- Make sure you have the Steam GFWL version of the game, as this is the only version this plugin is compatible with. (exe size of 4242504 bytes)
- Plop the files into your game folder.
- Enjoy, nya~ :3

## Known issues

- Cars sometimes spawn underground, restart the race to fix it
- Skidmarks and tire smoke no longer work
- Tires do not visually match at all

## Building

Building is done on an Arch Linux system with CLion and vcpkg being used for the build process. 

Before you begin, clone [nya-common](https://github.com/gaycoderprincess/nya-common), [nya-common-fouc](https://github.com/gaycoderprincess/nya-common-fouc) and [CwoeeMenuLib](https://github.com/gaycoderprincess/CwoeeMenuLib) to folders next to this one, so they can be found.

Required packages: `mingw-w64-gcc`

You should be able to build the project now in CLion.
